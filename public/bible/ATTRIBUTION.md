# Bible text attribution

The JSON files in this folder are public-domain English Bible texts adapted from:

[midvash/bible-data](https://github.com/midvash/bible-data)

| File | Translation | License |
|------|-------------|---------|
| `web.json` | World English Bible | Public domain |
| `kjv.json` | King James Version | Public domain (US) |
| `asv.json` | American Standard Version (1901) | Public domain |

Normalized to a compact schema for Scribe (book → chapters → verse text arrays).
