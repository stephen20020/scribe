# Bible text attribution

JSON files in this folder are normalized for Scribe (book → chapters → verse text arrays).

## English

Adapted from [midvash/bible-data](https://github.com/midvash/bible-data).

| File | Translation | License |
|------|-------------|---------|
| `web.json` | World English Bible | Public domain |
| `kjv.json` | King James Version | Public domain (US) |
| `asv.json` | American Standard Version (1901) | Public domain |

## Spanish

Modern commercial translations (RVR1960, NVI, DHH, etc.) are copyrighted and are not bundled. Scribe ships freely redistributable Spanish texts instead:

| File | Translation | Source | License |
|------|-------------|--------|---------|
| `rv1909.json` | Reina-Valera 1909 | [thiagobodruk/bible](https://github.com/thiagobodruk/bible) (`es_rvr`) | Public domain |
| `bll.json` | Biblia Libre Latinoamericana | [eBible.org `spabll`](https://ebible.org/Scriptures/spabll_usfm.zip) | Public domain |
| `onbv.json` | Open Nueva Biblia Viva | [eBible.org `spaonbv`](https://ebible.org/Scriptures/spaonbv_usfm.zip) | © Biblica, Inc. — freely redistributable open license |
